Overview

The Movie Information Retrieval (IR) System is a web-based application that allows users to search and retrieve movie-related documents from a collection of text files. It implements core information retrieval techniques like indexing, searching, and ranking to provide relevant movie information to users.

Usage

To run the application:

    Clone the repository
    Install dependencies and required libraries
    Run python main.py
    Access the application at http://localhost:5000

The homepage allows searching for movies by title, actor etc. Results can be sorted and filtered on the results page. Clicking a movie shows more details like description and poster.
